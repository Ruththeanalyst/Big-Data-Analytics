# Big-Data-Analytics
This repository contains hands-on labs completed on Big Data Analytics


## Course Coverage
- Cloud-based Jupyter environments
- Exploratory data analysis
- MapReduce using Python
- Apache Spark and PySpark
- Cloud data storage and autoscaling
- Spark clusters
- Business process analysis (FoodShiftr case)
- Stream processing
- Data privacy via remote analytics

## Technologies Used
- Python
- Jupyter Notebook
- Apache Spark
- PySpark
- Google Cloud platforms
- Kafka (stream processing)

## Purpose
This repository demonstrates practical experience in data analytics, big data processing, and cloud-based data systems.
